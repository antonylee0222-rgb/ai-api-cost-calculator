還應該補進商業模型的資訊

AI 成本不是只有 token 價格，這些欄位會決定你的產品是否真的能賺錢。

免費額度與濫用控管

設定免費用戶每月請求上限、每日尖峰限制、異常重試警報，避免免費流量吃掉毛利。

輸出長度與工具呼叫

模型回覆越長、agent 工具越多、重試越頻繁，成本會非線性上升。

快取、批次與降級策略

可快取的摘要、分類、模板化回答應該優先快取；非關鍵任務可改用便宜模型。

客戶分層定價

低價方案限制用量，高價方案提供高階模型、較長 context、優先佇列或團隊功能。

毛利安全線

早期至少保留 60% 以上毛利空間，因為之後還會加入客服、監控、資料庫與行銷成本。

單位經濟檢查

同時追蹤 CAC、LTV、退訂率與付費轉換率，避免只看 API 成本而忽略整體回本週期。
