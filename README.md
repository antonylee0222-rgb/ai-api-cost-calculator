# AI API 成本試算工具

給 AI 創業者與小型團隊免費使用的靜態試算器。

## 可以試算

- 每月 AI API 成本
- 每位活躍用戶與付費用戶成本
- 訂閱價格是否合理
- 變動毛利率與每月淨貢獻
- 需要多少付費用戶才能打平
- 便宜、均衡、高階模型方案比較
- 用量成長與 token 超量的敏感度

## 使用方式

直接開啟 `index.html`，或部署到 GitHub Pages、Netlify、Vercel 等靜態網站服務。

這個工具完全在瀏覽器端計算，不需要登入、不需要 API key，也不會把輸入資料送到伺服器。

## 重要限制

- 模型方案比較使用示範價格層級，不代表任何 provider 的官方即時價格。
- 「整體 token 成本節省率」會同時套用到輸入與輸出 token，實際上線前請依真實快取命中率與輸出縮減策略校正。
- 本工具適合初步估算與商業健檢，不應取代正式財務模型、稅務、退款、CAC、LTV 或法遵評估。

公式與測試案例請見 `VERIFICATION.md`。
